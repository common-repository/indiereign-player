== Installation ==

As Widget
---------

Step 1: Copy zip file in plugins folder

Step 2: Unzip file

Step 3: Open blog/site and login as admin

Step 4: Go to plugin section and activite the 'IRPlayer' Plugin

Step 5: Go to widget page and 'drag and drop' IRPlayer widget in your sidebar

Step 6: Fill the widget form 'Video Slug', 'Player Widget', and 'Player Height' to show player correctly

Step 7: Save change.


As Shortcode in post
--------------------

Do steps 1 to 4 as define above

Step 5: Open your post

Step 6: Now write the following line

[irplayer vslug="<VIDEO-SLUG>" width="<PLAYER-WIDTH>" height="<PLAYER-HEIGHT>"]

Step 7: Save your post. Now player is display on page.
